> 注意！GitHub Action 的排程極容易延誤，延誤時間通常是 10 到 60 分鐘，也有可能數個小時。如果你發現機器人沒有準時執行，請再等一段時間看看！
>
> 還在使用舊版機器人的使用者，請盡速更新至最新版（更新方式下方〈更新〉章節）。最新版將排程從早上 08:10 改為凌晨 00:10，這樣即使延誤也影響不大。
